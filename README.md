# Manufacturing Workshop Website

A modern, responsive website for a manufacturing workshop built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Clean, professional design with red and black color scheme
- Responsive layout that works on all devices
- Modern UI components with smooth animations
- Contact form with validation
- Optimized performance with Next.js App Router
- TypeScript for type safety
- Tailwind CSS for styling

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd manufacturing-workshop
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Project Structure

```
src/
```
